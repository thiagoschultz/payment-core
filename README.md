# 💳 Payment Core

Production-grade Payment Processing Microservice built with ASP.NET Core.

Designed using Clean Architecture, Event-Driven patterns and production-ready infrastructure — inspired by real-world fintech systems (Nubank / Stripe / Stone style architecture).

---

# 🚀 Features

- ✔ Payment Processing API
- ✔ Multi-tenant support (JWT-based tenant resolution)
- ✔ Idempotency Keys
- ✔ Outbox Pattern
- ✔ Saga Pattern
- ✔ Pix Simulation
- ✔ Event-Driven Architecture (RabbitMQ)
- ✔ Redis Cache
- ✔ Circuit Breaker (Polly)
- ✔ Rate Limiting
- ✔ OpenTelemetry Distributed Tracing
- ✔ Prometheus Metrics
- ✔ Grafana Dashboards
- ✔ Docker & Docker Compose
- ✔ Kubernetes Ready
- ✔ Load Test with k6
- ✔ Unit & Controller Tests (xUnit + Moq)
